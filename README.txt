# Dependencies Used
React: [axios](https://www.npmjs.com/package/axios)  
Server: [CORS Anywhere](https://www.npmjs.com/package/cors-anywhere)

# Running demo
1. Open first terminal window
cd helloworldreact/react
npm start
2. Open second terminal window
cd helloworldreact/server
node server
